# دليل الأسعار للموردين - Supplier Pricing Guide

## الميزة الجديدة

الآن كل مورد (Supplier) يمكنه:
- ✅ رؤية أسعاره الخاصة
- ✅ إضافة وتعديل الأسعار
- ✅ حذف الأسعار
- ✅ واجهة عربية كاملة

## كيفية الاستخدام

### للمورد (Supplier):

1. **تسجيل الدخول كمورد**
   - الدخول إلى حسابك

2. **الذهاب لصفحة "أسعاري"**
   - ستجد في القائمة الجانبية "My Pricing" أو "أسعاري"
   - اضغط عليها

3. **إضافة الأسعار**
   - اضغط على "إضافة سعر"
   - اختر نوع الخدمة:
     - **كوينز**: سعر المليون (مثال: 15 دولار للمليون)
     - **رتبة الفوت**: اختر الرتبة (1-6) وضع السعر
     - **رايفلز**: اختر الديفيجن (1-10) وضع السعر
     - **SBC - خدمة التشالنج**: سعر التشالنج الواحد

4. **تحديد المنصة**
   - PS (بلايستيشن)
   - PC (كمبيوتر)

5. **حفظ السعر**
   - اضغط "Save" أو "حفظ"

## أنواع الخدمات بالتفصيل

### 1. كوينز (Coins)
- **السعر**: بالدولار لكل مليون
- **مثال**: إذا وضعت 15 دولار:
  - طلب 100K = $1.50
  - طلب 500K = $7.50
  - طلب 1M = $15.00
- **ملاحظة**: نفس السعر يستخدم لطلبات SBC أيضاً

### 2. رتبة الفوت (FUT Rank)
- **السعر**: لكل رتبة (Rank 1, 2, 3, 4, 5, 6)
- **مثال**: Rank 3 = $50
- **خدمة مستعجلة**: يمكن وضع سعر إضافي للخدمة السريعة
- **مثال**: Rank 3 عادي = $50، Rank 3 سريع = $75

### 3. رايفلز (Rivals)
- **السعر**: لكل ديفيجن (Division 1-10)
- **مثال**: 
  - Division 1 = $100
  - Division 2 = $80
  - Division 3 = $60

### 4. SBC - خدمة التشالنج
- **السعر**: لكل تشالنج
- **مثال**: إذا وضعت 5 دولار:
  - 4 تشالنجات = $20
  - 8 تشالنجات = $40

## مثال عملي

**المورد "أحمد" يريد وضع أسعاره:**

```
1. كوينز PS: $15 للمليون
2. كوينز PC: $18 للمليون
3. FUT Rank 3 PS: $50
4. FUT Rank 3 PS (سريع): $75
5. Rivals Division 1 PS: $100
6. SBC التشالنج PS: $5
```

**النتيجة:**
- طلب 200K كوينز PS → $3 (15/1000 × 200)
- طلب Rank 3 عادي → $50
- طلب Rank 3 سريع → $75
- طلب Rivals Div 1 → $100
- طلب SBC 4 تشالنجات → $20 (5 × 4)
